# 环境要求

  • PHP 7.1+
  • Composer

# 安装

EasyDingTalk 是一个标准包，你可以使用 composer 进行安装

composer require mingyoung/dingtalk:^2.0

# 更新

要更新 EasyDingTalk，你可以使用以下命令进行更新

composer update mingyoung/dingtalk

# 实例化

$config = [
